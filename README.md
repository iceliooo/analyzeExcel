# Excel数据分析工具

## 项目概述

这是一个专门用于分析客户问题上报数据的Python工具集，主要用于统计和分析V8客户问题上报的各类指标，支持多维度数据分析和可视化报表生成。

## 主要功能

### 1. 多维度数据分析
- **按客户维度分析** (nalyzeExcel.py)
- **按开发人员维度分析** (nalyzeExcel-developer.py) 
- **按部门维度分析** (nalyzeExcel-group.py)
- **综合分析** (nalyzeExcel-combined.py) - 支持三个维度的综合分析

### 2. 核心统计指标

#### 基础统计量
- **总问题量** - 统计各维度的总问题数量
- **处理完成量** - 已完成处理的问题数量
- **有效问题量** - 被标记为有效的问题数量
- **缺陷问题量** - 被标记为缺陷的问题数量
- **开发超期量** - 开发处理超期的问题数量
- **一次交付量** - 一次交付成功的问题数量
- **严重问题量** - 被标记为严重的问题数量

#### 时间统计
- **有效问题总处理时长** - 有效问题的总处理时间（小时）
- **有效问题平均处理时长** - 有效问题的平均处理时间（小时）

#### 关键比率指标
- **有效率%** - 有效问题占总问题的百分比
- **缺陷率%** - 缺陷问题占总问题的百分比
- **处理及时率%** - 及时处理的问题占完成问题的百分比
- **一次交付率%** - 一次交付成功的问题占完成问题的百分比
- **严重问题率%** - 严重问题占总问题的百分比

### 3. 高级功能

#### 时间区间分析 (week/ 目录)
- 支持指定时间范围的数据分析
- 命令行参数支持，可自定义开始和结束日期
- 默认分析最近30天的数据

#### Excel报表生成
- 自动生成格式化的Excel分析报告
- 支持多工作表输出（按客户、按人员、按部门）
- 专业的表格样式和颜色标识
- 自动调整列宽和文本换行

#### 数据可视化
- 蓝色标题行标识
- 橙色关键指标列高亮
- 自动列宽调整
- 文本换行和对齐优化
